## Troubleshooting

**SSH Key Error?**
Use HTTPS instead:
```bash
git remote set-url origin https://github.com/tamilarasanmapla-sketch/e-com.git
git push origin tam
```

**Nothing to Commit?**
If git says "nothing to commit", your code is already up to date. Check with:
```bash
git status
git log --oneline -5
```

**Want to Push to Main Instead?**
```bash
git push origin tam:main
```
